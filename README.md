# NearHorizon

Personally, I felt a significant difference in FPS from reducing far clip.
Vanilla = 400 | mod default = 120

Based on [ViewExtension](https://github.com/sfDesat/ViewExtension)

Extra: FancyLamp + LungApparatus **light shadows off**

**Config**
- **`FarClipPlane`** — view distance.
- **`StripLampLungShadows`** — lamp/lung lights no shadows.
- **`DebugLogging`** — debug log.

**Caution**
- Another mod touching the same camera may win depending on load order.
- Some mod moons need `FarClipPlane` more than 400.
